Dungeon Generator

Games Programmer

Created for my final year masters project the dungeon generator allows the creation of multiple different styles of rooms.
The generator also places these rooms randomly within the dungeons dimensions.

To add more meat to the project the player can place and dig the tiles within the dungeon.

Some of these tiles drop items for the player to use in the crafting system.
